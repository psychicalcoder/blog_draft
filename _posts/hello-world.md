#### How to print "Hello World"

<!-- more -->

###### C/C++

``` C
#include<stdio.h>
int main() {
    puts("Hello World!");
    return 0;
}
```

###### Java

``` Java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}
```
